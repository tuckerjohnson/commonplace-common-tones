## Usage

Required Fonts:

- haydn ([OpenLilyPondFont](https://github.com/OpenLilyPondFonts/haydn))
- Junicode
- Syne

From a command prompt in the root directory:

`cat variables/* eng-files/* front-matter/* | lilypond -dno-point-and-click --output="Circumnutation"`
